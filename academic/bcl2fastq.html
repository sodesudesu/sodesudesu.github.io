How to install bcl2fastq

Home / Academic /

Illumina bcl2fastq is a BCL-to-FASTQ conversion tool. For more details, technical support page (?).

Method1: rpm to deb conversion

This is the most easy way.

illumina destributes only RPM package. Alien command convert .rpm to .deb and allow as to install bcl2fastq in Ubuntu.

Step1

Install alian 

sudo apt install alien

Step2

Cnvert RPM to DEB 

sudo alien bcl2fastq2-v*.**.*.**-Linux-x86_64.rpm

Step3

Extract DEB file 

sudo dpkg -i bcl2fastq2_0v*.**.*.**-*_amd64.deb

Method2: full compilation

Official document's description:

Step1

Download source files from https://jp.support.illumina.com/sequencing/sequencing_software/bcl2fastq-conversion-software/documentation.html (illumina id registration are required)

Step2

Extract .tar.gz file 

tar -xvzf bcl2fastq2-v*.**.x.tar.gz

Step3

Make build directory and move there 

mkdir ${BUILD}
cd ${BUILD}

Grant editing permission 

chmod ugo+x ${SOURCE}/src/configure
chmod ugo+x ${SOURCE}/src/cmake/bootstrap/installCmake.sh

mkdir ${INSTALL_DIR}
${SOURCE}/src/configure --prefix=${INSTALL_DIR}

(${SOURCE}: path to extracted source code)

Step4

At the same place, 

make
make install
